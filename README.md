#  проект **metricgen**

## Общее описание проекта

**metricgen** — распределённая система для генерации синтетических метрик в формате Prometheus TSDB, предназначенная для нагрузочного тестирования, симуляции инфраструктурных сценариев и проверки масштабируемости хранилищ временных рядов (Thanos, VictoriaMetrics, Mimir и др.). Проект включает оркестратор, пул воркеров, очереди задач (Kafka / in‑memory), хранилище в S3 (блоки, схемы, метаданные задач и тренировочные данные), Web UI с дашбордом и калькулятором, а также полную наблюдаемость через Prometheus-метрики.

---

## Назначение системы

| Ключевая задача | Решение |
|----------------|---------|
| Генерация синтетических данных для Prometheus/Thanos | TSDB-блоки, совместимые с Thanos Store Gateway |
| Масштабирование до тысяч хостов и миллионов серий | Горизонтальное масштабирование воркеров через Kafka + KEDA |
